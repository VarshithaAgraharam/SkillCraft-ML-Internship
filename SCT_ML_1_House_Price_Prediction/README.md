# House Price Prediction

## Objective
Build a machine learning model to predict house prices based on various housing features.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-Learn
- Matplotlib
- Seaborn

## Model
- Linear Regression

## Dataset
- Housing Dataset

## Results
The model was trained using Linear Regression and evaluated using:
- Mean Absolute Error (MAE)
- Mean Squared Error (MSE)
- Root Mean Squared Error (RMSE)
- R² Score

## Outputs
- Correlation Heatmap
- Actual vs Predicted Plot
- Residual Distribution Plot
- Sample Predictions

## Conclusion
This project demonstrates the application of Linear Regression for predicting house prices using housing-related features.
