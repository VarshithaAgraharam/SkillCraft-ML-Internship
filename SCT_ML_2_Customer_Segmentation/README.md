# Customer Segmentation

## Objective
Segment customers into different groups based on purchasing behavior using machine learning techniques.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-Learn
- Matplotlib
- Seaborn

## Model
- K-Means Clustering

## Dataset
- Online Retail Dataset

## Results
The dataset was analyzed and customers were grouped into different segments based on their purchasing patterns.

## Outputs
- Elbow Method Graph
- Customer Segmentation Visualization
- Cluster Summary

## Conclusion
This project demonstrates how K-Means Clustering can be used to identify customer groups and uncover meaningful purchasing patterns.
