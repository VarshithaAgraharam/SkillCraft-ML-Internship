# Cat vs Dog Classification

## Objective
Build an image classification model to distinguish between cat and dog images using machine learning techniques.

## Technologies Used
- Python
- NumPy
- OpenCV
- Scikit-Learn
- Matplotlib
- Seaborn

## Model
- Support Vector Machine (SVM)
- RBF Kernel

## Dataset
- Microsoft Cats vs Dogs Dataset

## Methodology
1. Loaded cat and dog images from the dataset.
2. Resized images for efficient processing.
3. Normalized image pixel values.
4. Split the dataset into training and testing sets.
5. Trained an SVM classifier using the RBF kernel.
6. Evaluated the model using accuracy and classification metrics.

## Results
The model successfully classified cat and dog images using Support Vector Machine (SVM).

### Performance
- Test Accuracy: ~59%

## Outputs
- Confusion Matrix
- Sample Prediction
- Model Performance Report

## Conclusion
This project demonstrates the application of machine learning techniques for image classification. The model was able to identify patterns in image data and classify images into cat and dog categories.
