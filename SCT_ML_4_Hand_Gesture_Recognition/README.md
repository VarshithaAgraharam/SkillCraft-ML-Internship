# Hand Gesture Recognition

## Objective
Develop a deep learning model to recognize and classify hand gestures from images using Convolutional Neural Networks (CNN).

## Technologies Used
- Python
- TensorFlow
- Keras
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn

## Model
- Convolutional Neural Network (CNN)

## Dataset
- Hand Gesture Recognition Dataset
- 20 Gesture Classes
- 18,000 Training Images
- 6,000 Testing Images

## Methodology
1. Loaded and preprocessed image data.
2. Rescaled image pixel values.
3. Built a CNN model using convolutional and pooling layers.
4. Trained the model on the training dataset.
5. Evaluated model performance on the testing dataset.
6. Generated performance metrics and visualizations.

## Results
The CNN model successfully learned hand gesture patterns and achieved excellent classification performance.

### Performance
- Number of Classes: 20
- Training Images: 18,000
- Testing Images: 6,000
- Test Accuracy: ~100%

## Outputs
- Accuracy Curve
- Loss Curve
- Confusion Matrix
- Sample Prediction
- Model Performance Report

## Conclusion
This project demonstrates the effectiveness of Convolutional Neural Networks (CNNs) for image classification tasks. The model successfully recognized and classified hand gestures across 20 different categories with high accuracy.
