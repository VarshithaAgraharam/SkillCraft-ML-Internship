# SkillCraft ML Internship Projects

This repository contains a collection of Machine Learning and Deep Learning projects completed as part of the **SkillCraft Technology Machine Learning Internship**. The projects demonstrate practical applications of supervised learning, unsupervised learning, computer vision, and deep learning using Python and popular machine learning libraries.

Through these projects, I gained hands-on experience in data preprocessing, exploratory data analysis, model development, performance evaluation, and image classification techniques.

---

## Projects Included

### 1. House Price Prediction

Developed a Machine Learning model to predict house prices using various housing attributes. The project involved data cleaning, feature encoding, correlation analysis, model training, and performance evaluation using Linear Regression.

#### Key Highlights
- Data preprocessing and feature engineering
- Correlation analysis using heatmaps
- Linear Regression model implementation
- Performance evaluation using MAE, MSE, RMSE, and R² Score

#### Technologies Used
- Python
- Pandas
- NumPy
- Scikit-Learn
- Matplotlib
- Seaborn

---

### 2. Customer Segmentation

Implemented K-Means Clustering to segment customers based on purchasing behavior. The project helps identify different customer groups and provides insights into customer patterns.

#### Key Highlights
- Customer data preprocessing
- Feature scaling using StandardScaler
- Elbow Method for optimal cluster selection
- Customer segmentation visualization

#### Technologies Used
- Python
- Pandas
- NumPy
- Scikit-Learn
- Matplotlib
- Seaborn

---

### 3. Cat vs Dog Classification

Built an image classification model to distinguish between cat and dog images. Image preprocessing techniques were applied before training an SVM classifier.

#### Key Highlights
- Image preprocessing and resizing
- Feature extraction
- Support Vector Machine (SVM) classification
- Confusion Matrix and performance evaluation

#### Technologies Used
- Python
- OpenCV
- NumPy
- Scikit-Learn
- Matplotlib
- Seaborn

---

### 4. Hand Gesture Recognition

Developed a Convolutional Neural Network (CNN) to recognize and classify hand gestures across multiple categories. This project demonstrates the application of deep learning techniques in computer vision tasks.

#### Key Highlights
- Multi-class image classification
- CNN model development
- Model training and evaluation
- Accuracy and loss visualization
- Gesture prediction and performance analysis

#### Technologies Used
- Python
- TensorFlow
- Keras
- NumPy
- Matplotlib
- Seaborn
